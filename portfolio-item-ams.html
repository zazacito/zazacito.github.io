The Development of an Athlete Management System

Biarritz Olympique Pays Basque

I worked as a full stack developer then as a data scientist for the professional rugby club of Biarritz Olympique Pays Basque. My main mission was the development of a web application for sports performance management (Athlete Management System) and then the exploitation of this application with the professional team.

The main objective was the development of a web application allowing staff to read and write key data and key performance indicators (KPI) at any time.

I developed a web application using React JS hosted on Firebase Hosting. The database used is a No SQL database named Firestore, linked to the application via a Serverless API in Node JS.

Staff members could access and modify at any time individual player profiles to consult their gps reports, training loads, physical testing score, well-being questionaires or player individualised training routine.

Coaching staff could also consult collective performance reports like, for example, weekly performance totals for each position.

In addition, a system of notifications has been added to notify the staff of important changes in players fitness for example.
