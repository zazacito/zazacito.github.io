The Development of a Web Application for social and sports activities

By The Wave Technologies

I worked as a web developer at BYTHEWAVE TECHNOLOGIES, a small start-up specialized in technological innovation linked to sliding sports. My mission was to develop a web application specifically designed for disabled surfers. The aim of the application is to work as a contact facilitator for surfers who are part of the association “Handi Surf”.

I created the application using such as HTML5, CSS and JavaScript for the front-end part, for the back-end part, I also used JavaScript in addition to the firebase console.
